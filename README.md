# AI-SOP Platform

AI-SOP is a comprehensive, AI-powered Standard Operating Procedure (SOP) compliance and audit management platform designed for FDA/GxP regulated industries. 

It provides an end-to-end authoring, reviewing, and approval pipeline ensuring high standards of data integrity and compliance.

## Sub-projects

* `frontend`: The React / Vite user interface.
* `backend`: The Python / Flask API and AI orchestration server.
